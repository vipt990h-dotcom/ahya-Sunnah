منصة إحياء سنة V10

محتويات الحزمة:
- index.html : الصفحة الرئيسية للمنصة.
- assets/logo.jpeg : أيقونة وشعار إحياء سنة.
- assets/badge_*.svg : ملفات الأوسمة.
- certificates/certificate_template.png : صورة قالب شهادة سفير إحياء السنة.
- certificates/certificate.html : صفحة مستقلة للشهادة مع إدخال اسم المشاركة والطباعة.
- manifest.json : ملف تثبيت المنصة كتطبيق.
- service-worker.js : ملف خدمة بسيط.
- css/style.css و js/app.js : ملفات تنظيمية إضافية.
